# 心选倒计时 - 一款简约优雅的倒计时应用

## 项目简介

**心选倒计时（FocusCount）** 是一款基于 UniApp + Vue3 开发的跨平台倒计时应用，支持 H5 网页端和微信小程序。采用清新蓝简约设计风格，专注于记录生活中的重要时刻。

## 功能亮点

### 首页沉浸式展示
- 单项目全屏倒计时，秒级实时更新
- 天数大卡片 + 时分秒小卡片，视觉层次分明
- 7 天内自动红色预警提醒
- 到期自动显示「已结束」状态

### 灵活的项目管理
- 支持 5 种类型：考试、项目截止、纪念日、生日、自定义
- 7 种颜色标签可选，视觉区分不同项目
- 星标置顶功能，首页专注展示最重要的事项

### 便捷的交互体验
- 新增/编辑均使用居中弹窗，无需页面跳转
- 卡片悬浮菜单，操作一触即达
- 支持当天、提前 1/3/7 天提醒设置
- 纪念日和生日自动支持每年重复

## 技术栈

| 技术 | 说明 |
|------|------|
| UniApp | 跨平台框架，一套代码多端运行 |
| Vue 3 | Composition API，响应式开发 |
| Vite | 下一代前端构建工具 |
| TypeScript | 类型安全，提升代码质量 |
| Pinia | Vue 官方状态管理 |
| SCSS | CSS 预处理器，样式模块化 |

## 项目结构

```
src/
├── components/     # 公共组件
├── pages/          # 页面
│   ├── index/      # 首页（置顶展示）
│   └── list/       # 全部倒计时列表
├── services/       # 业务逻辑层
├── stores/         # Pinia 状态管理
├── styles/         # 全局样式
├── types/          # TypeScript 类型定义
└── utils/          # 工具函数
```

## 设计理念

### 分层架构
```
页面/组件（UI 渲染）
    ↓
Store（状态管理）
    ↓
Service（业务逻辑）
    ↓
Utils/Types（工具 + 类型）
```

页面组件只负责 UI 渲染和用户交互，业务逻辑封装在 Service 层，通过 Store 统一管理状态，职责清晰，易于维护。

### 视觉设计
- **主色调**：清新蓝 `#3B82F6`
- **背景色**：柔和薰衣草蓝 `#E8EEF8`
- **圆角卡片**：白色背景 + 柔和阴影
- **字体层次**：天数超大字号（160rpx）+ 时分秒中等字号（56rpx）

## 快速开始

```bash
# 安装依赖
npm install

# H5 开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin

# 构建 H5
npm run build:h5

# 构建微信小程序
npm run build:mp-weixin
```

## 多端打包

| 平台 | 命令/方式 | 说明 |
|------|----------|------|
| H5 | `npm run build:h5` | 部署到任意 Web 服务器 |
| 微信小程序 | `npm run build:mp-weixin` | 微信开发者工具上传 |
| Android APK | HBuilderX 云打包 | 需 DCloud 账号 |
| iOS IPA | HBuilderX 云打包 | 需 Apple 开发者账号 |
