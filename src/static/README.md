# 静态资源说明

## 需要替换的图标文件

以下图标文件为占位文件，需要替换为实际的设计图标：

### 1. tab-home.png
- **用途**: 底部Tab栏"首页"图标
- **建议尺寸**: 48x48px（@1x）、96x96px（@2x）、144x144px（@3x）
- **建议格式**: PNG（透明背景）
- **设计要求**: 时钟/倒计时风格图标，未选中为灰色(#909399)，选中为主题色(#4A6CF7)

### 2. tab-list.png
- **用途**: 底部Tab栏"全部"图标
- **建议尺寸**: 48x48px（@1x）、96x96px（@2x）、144x144px（@3x）
- **建议格式**: PNG（透明背景）
- **设计要求**: 列表风格图标，未选中为灰色(#909399)，选中为主题色(#4A6CF7)

## 注意事项

- 图标应保持统一的视觉风格
- 建议使用矢量图导出，确保各尺寸清晰
- 选中态和未选中态可分别导出，也可通过代码 tint 实现颜色切换
